# ChongChong Tabs Downloader

Download tabs with one command.

## Supported Sites

- www.gangqinpu.com (虫虫钢琴网)

## Usage

### Download

```
python downloader.py <some_url>
```

### Convert image

```
python piano_convert.py -i <input_dir> -o <output_dir>
```
